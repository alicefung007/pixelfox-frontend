# Project Checklist: pixelfox Implementation

## 1. Core UI Components
- [ ] Navbar (Editor, Gallery, Upscaler, Profile, Notifications, Language, Theme)
- [ ] Sidebar (Action buttons, History, Export/Clear, Settings, Tool panel, Canvas size)
- [ ] Main Editor Area (Canvas, Zoom controls)
- [ ] Palette Area (Color tabs, Color swatches, Recent/Used colors)

## 2. Drawing Tools & Logic
- [ ] Brush (Individual pixel drawing)
- [ ] Bucket (Flood fill tool)
- [ ] Hand (Pan tool)
- [ ] Eraser (Remove pixel color)
- [ ] Eyedropper (Pick color from canvas)
- [ ] Text (Add text layer/pixels)

## 3. Features
- [ ] Multi-language (English/Chinese toggling)
- [ ] Dark/Light mode theme switching
- [ ] Undo/Redo (History management)
- [ ] Canvas Zoom (Zoom in/out, percentage display)
- [ ] Export (Mock export functionality)

## 4. Technical Requirements
- [ ] React 19 + TypeScript
- [ ] shadcn-ui + Tailwind CSS
- [ ] Zustand for global state management
- [ ] React Router for page navigation
- [ ] Clean, maintainable component architecture

## 5. Visual Consistency
- [ ] Vibrant pink/purple gradients for key UI elements
- [ ] Precise iconography (Lucide or Hugeicons)
- [ ] Layout matches the reference image (Sidebar width, Navbar height, etc.)
- [ ] Smooth interactions and hover states
